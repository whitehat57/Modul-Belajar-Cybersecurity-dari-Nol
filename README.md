# 🛡️ Cybersecurity dari Nol — Modul Belajar 3 Bulan

Modul belajar cybersecurity terstruktur untuk pemula yang ingin masuk ke dunia keamanan siber secara sistematis — dari fondasi hingga siap karier profesional.

> Dibuat sebagai panduan awal yang padat. Setiap topik dirancang untuk dikembangkan dan diperdalam lebih lanjut secara mandiri.

## ⚠️ Etika & Disclaimer

Semua materi di repositori ini ditujukan **murni untuk edukasi**.

- ✅ Praktikkan hanya di lab pribadi, VM, atau platform resmi (TryHackMe, HTB, DVWA)
- ✅ Bug bounty hanya pada program yang memiliki izin eksplisit
- ❌ Jangan gunakan ilmu ini untuk menyerang sistem tanpa izin
- ❌ Hacking tanpa izin adalah tindak pidana di hampir semua yurisdiksi, termasuk Indonesia (UU ITE)

## 🚀 Cara Memulai

**1. Clone repositori**
```bash
git clone https://github.com/username/repo-name.git
cd repo-name
```

**2. Buka modul**

Bisa dibaca langsung di GitHub, atau gunakan tools Markdown favoritmu:
- [Obsidian](https://obsidian.md) — direkomendasikan untuk catatan pribadi
- [VSCode](https://code.visualstudio.com) + extension Markdown Preview
- Notion (import Markdown)

**3. Ikuti urutan**

Mulai dari Minggu 1 dan jangan loncat. Fondasi di Bulan 1 adalah kunci untuk memahami Bulan 2 dan 3.
